## Install 

Ensure you've [Node.js](https://nodejs.org) version >=4 installed, then run the following:

```
$ [sudo] npm install mzsi-cli --global 
```

## Usage
```
	Usage:
    	mzsi <month> <day>
      
  	Example:
    	mzsi 07 07
```
